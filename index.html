一、引言：use 不是客户端数据请求工具

React 19 引入的 use API 是一个在渲染期间读取 Promise 或 Context 的原语。在 Next.js 等 RSC 架构下，典型用法是：Server Component 创建但不 await Promise，Client Component 用 use(promise) 挂起等待——这样 Server Component 本身不会被阻塞，页面可以更早开始流式输出。
